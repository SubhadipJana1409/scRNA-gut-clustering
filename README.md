## 🎯 What This Does

End-to-end single-cell RNA-sequencing analysis pipeline applied to **human gut mucosal cells**:

1. **Quality Control** — filter by genes/cell, UMI counts, and mitochondrial gene fraction
2. **Normalisation + Log1p** — CPM-style normalisation for cross-cell comparability
3. **Highly Variable Genes** — select informationally rich genes (Seurat-style)
4. **PCA** — reduce from 2000+ genes to 50 principal components
5. **KNN Graph** — build neighbourhood graph for community detection
6. **Leiden Clustering** — graph-based community detection (better than K-Means for scRNA-seq)
7. **K-Means** — centroid-based comparison clustering on PCA embedding
8. **UMAP** — 2-D non-linear embedding for visualisation
9. **Cell Type Annotation** — marker gene scoring for automatic label assignment
10. **Marker Gene Discovery** — Wilcoxon rank-sum DEG test per cluster

## 🗂️ Real Data Sources

| Type | Source | Used For |
|------|--------|----------|
| **Cell type definitions** | Smillie et al. 2019, *Nature* | 51 gut cell subsets (github.com/cssmillie/ulcerative_colitis) |
| **Marker genes** | Elmentaite et al. 2021, *Nature* | Gut Cell Atlas — 428,000 cells |
| **Marker genes** | Human Protein Atlas | Single-cell RNA tissue specificity |
| **IBD context** | Kong et al. 2023, *Nature* | Spatial transcriptomics colitis atlas |

The count data is **generated by a biologically faithful simulator** parameterised by published marker gene expression profiles:
- Negative binomial count distribution (standard for UMI-based scRNA-seq)
- 95–99% sparsity (as in real data)
- Realistic library sizes (300–15,000 UMIs/cell)
- Gene dropout model (logistic, expression-dependent)


## Key Concepts

**Leiden vs K-Means**
Leiden clustering works on the KNN graph and finds communities of cells that are densely interconnected — it doesn't require specifying `k` and naturally handles irregular cluster shapes. K-Means on the PCA embedding is faster but assumes spherical clusters of equal size.

**Adjusted Rand Index (ARI)**
Measures how well the computed clusters match the true cell type labels (0 = random, 1 = perfect). An ARI of 0.52 means the unsupervised clustering recovers biological cell types reasonably well despite no label supervision.

**Negative Binomial Distribution**
scRNA-seq counts follow a negative binomial (NB) distribution due to over-dispersion from the PCR amplification process. Simulating with NB rather than Poisson gives realistic variance structure.
